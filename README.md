# Akıllı Kişiselleştirilmiş Haber Asistanı

Bu proje, akıllı bir haber asistanı uygulamasıdır.

## Kurulum

1. Gereksinimleri yükleyin:
   pip install -r requirements.txt

2. .env dosyasını .env.example'dan kopyalayın ve ayarlayın.

3. Veritabanını başlatın:
   python src/database/setup.py
